# 命令行翻译工具

1、安装
```
npm i -g hhw-node-fanyi
```

2、使用
```
fy <word>
```

## 实现
#### 1、使用commander接收命令行参数

#### 2、百度翻译api

#### 3、https.request(options, callback)发送请求

#### 4、发布typescript包

1. 新增tsconfig.json `tsc --init`
2. 编辑tsconfig.json
```
"outDir": "dist/", 
```
3. 编译typescript文件 `tsc`
